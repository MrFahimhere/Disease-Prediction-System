# 🩺 Disease Predictor using Machine Learning

A Streamlit-based web application that predicts possible diseases based on user-selected symptoms using a Random Forest Classifier trained on a disease-symptom dataset.

## 📌 Overview

This project uses Machine Learning to predict diseases from symptoms entered by the user. The model is trained on a dataset containing diseases and their associated symptoms. Users can select symptoms through an interactive Streamlit interface and receive an instant prediction along with the model's confidence score.

---

## ✨ Features

* Disease prediction based on selected symptoms
* Interactive and modern Streamlit UI
* Symptom search and filtering
* Confidence score for predictions
* Model accuracy display
* Confusion matrix visualization

## ⚙️ How It Works

### Data Preprocessing

* Removes empty columns from the dataset.
* Combines symptom columns into a single symptom list.
* Encodes symptoms using MultiLabelBinarizer.
* Encodes disease labels using LabelEncoder.

### Model Training

A Random Forest Classifier is trained using:

```python
RandomForestClassifier(
    n_estimators=100,
    random_state=42
)
```

### Evaluation

The model is evaluated using:

* Accuracy Score
* Confusion Matrix

### Model Saving

The following artifacts are saved:

* model.pkl
* mlb.pkl
* lb.pkl
* accuracy.pkl
* symptoms.pkl

These files are later loaded by the Streamlit application.

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/MrFahimhere/Disease-Prediction-System.git
cd disease-predictor
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🧠 Train the Model

Run:

```bash
python model.py
```

This will:

* Train the model
* Generate confusion_matrix.png
* Save all required model files

---

## 🌐 Run the Streamlit Application

```bash
streamlit run app.py
```

The application will open in your browser.

## 📊 Model Output

The application displays:

* Number of Symptoms Selected
